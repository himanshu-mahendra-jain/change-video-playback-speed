Change video and audio playback speed with one click or a keyboard shortcut.

Change Video Playback Speed is a lightweight, open-source extension that cycles through four useful playback speeds:

1x → 1.25x → 1.5x → 2x → 1x

Click the extension icon or use the keyboard shortcut to move to the next speed. The selected speed is applied to video and audio on the page, including supported embedded players.

The current speed is briefly shown on the page and displayed in the toolbar badge.

KEY FEATURES

• One-click and keyboard shortcut control
• Video and audio support
• Support for compatible embedded players
• Toolbar speed badge and on-page indicator
• No popup menus, floating controllers, or configuration

DEFAULT KEYBOARD SHORTCUTS

• Windows, Linux, and ChromeOS: Alt+S
• macOS: Command+Shift+Period

CHANGE THE KEYBOARD SHORTCUT

1. Open Chrome and go to Extensions → Manage Extensions.
2. Open the Extensions menu and select Keyboard shortcuts.
3. Find Change Video Playback Speed and enter your preferred shortcut.

PRIVACY

• No analytics or tracking
• No data collection or browsing history collection
• No network requests
• No permanent access to websites

All playback-speed changes happen locally in your browser.
